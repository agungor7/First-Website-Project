The World Wide Web

Web is composed of mostly HTML documents. Simply, you as a user send a Hypertext Transfer Protocol (HTTP) request to the server and the server sends you back the Hypertext Markup Language (HTML) document.

Basic HTML elements and attributes

HTML elements are the things that we tell to a browser how to display content on a web page. Some elements are OK with having additional values such as an anchor tag which gives us opportunity to add a URL to link to. We call these addional values attributes.
